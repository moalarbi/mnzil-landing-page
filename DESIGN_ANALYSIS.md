# تحليل موقع منزل - mnzil.com/ar

## الألوان الأساسية
- **اللون الأساسي**: أزرق عميق (#1e40af / oklch(0.4 0.15 260))
- **خلفية الموقع**: أبيض فاتح جداً (oklch(0.985 0.002 247.839))
- **نص أساسي**: رمادي داكن (rgb(34, 34, 34))
- **اللون الثانوي**: برتقالي (#FF8C00) - أزرار الـ CTA

## الخطوط المستخدمة
- **الخط الأساسي**: htEquilibra مع DM Sans كبديل
- **الخط الاحتياطي**: sans-serif

## البنية الأساسية للصفحة
1. **الرأس (Header)**: ملاح ثابت مع شعار وقوائم وأزرار
2. **البطل (Hero Section)**: 
   - خلفية بتصميم موجات زرقاء متحركة (أنيميشن)
   - عنوان رئيسي: "حل واحد، يغطي كل احتياجات عاملينك"
   - زر CTA برتقالي "ابدأ الآن"
   - معرض صور بـ 5 صور بإطارات زرقاء وحدود منقطة

3. **قسم الشركاء**: شركات موثوقة مع شعاراتها
4. **قسم الفروقات**: صور وميزات
5. **الفوتر (Footer)**: روابط وشبكات اجتماعية

## الأنيميشن الرئيسية
- خلفية موجات متحركة في Hero Section
- تأثيرات Hover على الأزرار والعناصر التفاعلية
- حركات دخول سلسة للعناصر

## الخصائص الأخرى
- الموقع يدعم RTL (اتجاه اليمين لليسار)
- تصميم متجاوب
- نمط حديث وعصري
